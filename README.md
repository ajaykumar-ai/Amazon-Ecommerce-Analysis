# 🛒 Amazon E-Commerce Sales Analysis

## 📌 Overview

Analyzed Amazon E-Commerce sales data using **SQL, Power BI, and Excel** to uncover insights on revenue growth, product performance, customer behavior, delivery efficiency, and return patterns.

## 🛠️ Tools Used

* SQL
* Power BI
* Excel
* DAX

## 📊 Key Insights

* Revenue exceeded **₹107M**
* Processed **113K+ orders**
* Phones & Tablets generated the highest revenue
* Higher product ratings positively impacted sales
* Express Delivery achieved the fastest delivery times
* Product quality issues were the leading return reason

## 💡 Recommendations

* Invest more in high-performing categories
* Improve product quality control
* Optimize international shipping operations
* Promote underperforming products strategically
